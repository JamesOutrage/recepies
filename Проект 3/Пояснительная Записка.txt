Название проекта
Кулинарная книга

Для чего нужен проект
Проект нужен, чтобы удобно хранить кулинарные рецепты и делиться ими с другими.

Как работает проект
Пользователь заходит на сайт, видит список всех рецептов. Может добавить свой рецепт, указав название, ингредиенты, инструкцию, время готовки и сложность. 
Может найти нужный рецепт через поиск. Открыв рецепт, можно поставить ему оценку и написать комментарий. Если рецепт больше не нужен, его можно удалить. 
Есть кнопка случайного рецепта, которая помогает выбрать блюдо, если не знаешь, что приготовить.

Какие функции реализованы
Просмотр всех рецептов. Поиск. Добавление рецептов. Просмотр одного рецепта. Оценка рецептов. Комментарии. Удаление рецептов. Случайный рецепт.

Сложности при разработке
Главная сложность для меня была — правильно связать базу данных с веб-страницами, чтобы данные сохранялись и отображались правильно. Решено с помощью SQLite.

Вывод
Проект получился полезным и полностью рабочим. Его можно использовать для хранения своих рецептов.
