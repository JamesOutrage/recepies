ТЗ: Кулинарная книга

Что это?
Сайт, где можно добавлять рецепты, ставить оценки и писать комментарии под рецептами

Что умеет сайт?
1. Показать все рецепты на главной странице.
2. Найти рецепт по названию или ингредиентам.
3. Добавить новый рецепт через форму.
4. Открыть страницу с полным рецептом.
5. Оценить рецепт звёздами от 1 до 5.
6. Написать комментарий под рецептом.
7. Удалить рецепт.
8. Получить случайный рецепт по кнопке.

Из чего состоит?
- Python + Flask (сервер)
- SQLite (база данных)
- HTML + CSS (внешний вид)
